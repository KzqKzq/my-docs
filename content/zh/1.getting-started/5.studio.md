## **介绍**

[Nuxt Studio](https://nuxt.studio) **Web 编辑器**是一个基于浏览器的可视化界面，用于创建、编辑和查看您的文档。它提供预览体验，同时使您的工作与 Git 存储库保持同步。

::prose-tip{to="https://content.nuxt.com/studio"}
在 Nuxt Content 文档中了解有关 Nuxt Studio 的更多信息。
::

## **Web 编辑器与 CLI 的比较**

[Nuxt Studio](https://nuxt.studio) 的 **Web 编辑器**允许您完全从浏览器管理文档。无需本地开发工具或终端命令。对于在一个集中的地方维护文档，使用一个无需任何 Markdown 技能的简单工具来说是理想的。

另一方面，**CLI（命令行界面）**是一个为喜欢在自己的 IDE 中工作的开发人员设计的本地工具。

::prose-note
两个工具都与 Git 完全集成，因此您可以根据需要在它们之间切换。团队成员可以选择最适合他们工作流的方法。
::

## **两个不同的编辑器**

Nuxt Studio 为开发人员和内容作者提供了一个通用的工作区，让他们可以自由地在两个不同的编辑器之间进行选择来进行内容创建和管理：**Markdown 编辑器**和**可视化编辑器**。

您可以从项目的设置页面选择您喜欢的编辑器。

::prose-note
每个编辑器都有其自己的用途，有些用户习惯于 Markdown 编辑，而其他用户更喜欢非技术性的可视化方法。最后，**两个编辑器的最终输出都是 Markdown 语法**。
::

## **Markdown 编辑器**

![开发人员的 Markdown 编辑器](/documentation/markdown-editor-studio.png)

Nuxt Studio 中的 Markdown 编辑器提供对内容的完全控制，允许您直接使用 **Markdown** 编写文档，并使用 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax)集成 Vue 组件。

当您使用 Markdown 编辑器保存文件时，内容会按照您编写的方式存储，保留所有特定的语法和格式。此编辑器适合熟悉 Markdown 的用户，他们希望对内容的布局和结构进行精确控制。

## **可视化编辑器**

![为非技术用户提供的可视化编辑器](/documentation/visual-editor-studio.png)

Nuxt Studio 编辑器受到 Notion 的启发，以其直观的设计和灵活性而闻名。就像标准文本编辑器一样，Studio 编辑器的设计目的是熟悉和易于使用。

然而，它以其增强编写体验的附加功能而脱颖而出：

### **工具栏**

突出显示文本以显示工具栏，使您可以访问 [Markdown 语法](/zh/essentials/markdown-syntax)提供的所有标准文本编辑功能：

- 标题格式
- 粗体
- 斜体
- 删除线
- 代码
- 链接
- 类
- 项目符号列表
- 编号列表

### **`/` 快捷方式**

只需在编辑器中的任何地方键入 `/` 即可访问所有 Studio 功能。

#### **格式化功能**

- 标题格式
- 换行
- 水平线
- 代码块
- 段落
- 粗体和斜体

#### **组件**

Studio 的一个突出特点是它能够直接在编辑器中集成和自定义任何复杂组件。

换句话说，所有 [Nuxt UI 组件](/zh/essentials/components)都可以使用，并且可以直接从编辑器中集成。编辑器还可以调整组件属性、插槽和样式。

::prose-note
您还可以创建自定义组件并让用户从可视化编辑器中集成它们。
::

只需键入 `/` 即可访问所有可用组件的列表。

#### **图像**

使用 `/` 快捷方式，您可以通过选择"图像"选项快速插入图像。将打开一个模态框让您选择要插入的媒体。

从媒体模态框中，您可以为 SEO 和可访问性目的设置 **alt 属性**。
