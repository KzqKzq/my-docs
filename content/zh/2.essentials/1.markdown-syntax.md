## 标题

使用标题来介绍主要部分。它们构造您的文档并帮助用户导航内容。

::code-preview
---
class: "[&>div]:*:my-0"
---
## 标题

#code
```mdc
## 标题
```
::

### 副标题

使用副标题进一步划分部分。它们为更好的可读性创建了更详细的内容层次结构。

::code-preview
---
class: "[&>div]:*:my-0"
---
### 副标题

#code
```mdc
### 副标题
```
::

::tip
每个标题和副标题都创建一个锚点，并自动显示在目录中。
::

## 文本格式

Docus 支持大多数 Markdown 格式选项。

| 样式  | 如何使用   | 结果     |
| ------ | ------------ | ---------- |
| 粗体   | `**粗体**`   | **粗体**   |
| 斜体 | `*斜体*`   | *斜体*   |
| 删除线 | `~~删除线~~` | ~~删除线~~ |

结合格式以获得更丰富的文本样式和视觉强调。

| 样式         | 如何使用          | 结果            |
| ------------- | ------------------- | ----------------- |
| 粗体斜体   | `**_粗体斜体_**` | ***粗体斜体*** |
| 粗体删除线   | `~~**粗体**~~`      | ~~**粗体**~~      |
| 斜体删除线 | `~~*斜体*~~`      | ~~*斜体*~~      |
