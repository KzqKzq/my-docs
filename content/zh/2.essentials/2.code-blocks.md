## 基础

### 内联代码

使用内联代码在文本段落中显示代码片段。它非常适合在句子中直接引用代码元素。

::code-preview
---
class: "[&>div]:*:my-0"
---
`内联代码`

#code
```mdc
`内联代码`
```
::

### 代码块

使用代码块显示具有语法高亮的多行代码片段。代码块对于清楚地呈现代码示例至关重要。

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

#code
````mdc
```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
````
::

编写代码块时，您可以指定一个文件名，它将显示在代码块的顶部。将根据扩展名或名称自动显示图标。
文件名帮助用户理解代码在项目中的位置和目的。

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

#code
````mdc
```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```
````
::

当编写代码块时，您可以使用 `bash`、`shell`、`ts`、`js`、`python` 等多种语言。

::code-preview
---
class: "[&>div]:*:my-0 [&>div]:*:w-full"
---
```bash [Terminal]
npm run dev
```

#code
````mdc
```bash [Terminal]
npm run dev
```
````
::
