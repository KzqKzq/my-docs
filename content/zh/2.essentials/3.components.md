Prose 组件是 HTML 排版标签的替代品。它们提供了一种简单的方式来在使用 Markdown 时自定义您的 UI。

**Docus 和 Nuxt UI** 提供了一组样式美观的 prose 组件，以帮助您使用 [MDC 语法](https://content.nuxt.com/docs/files/markdown#mdc-syntax)编写文档。

::prose-note{to="https://ui.nuxt.com/getting-started"}
本页只强调了最适合编写文档的 prose 组件。但是，您可以在 Markdown 中使用任何 **Nuxt UI 或 Nuxt UI 组件**。有关可用组件的完整列表，请访问 Nuxt UI 文档。
::

### `Accordion`

使用 `accordion` 和 `accordion-item` 组件在内容中显示 [Accordion](https://ui.nuxt.com/components/accordion)。

  :::tabs-item{icon="i-lucide-eye" label="预览"}
