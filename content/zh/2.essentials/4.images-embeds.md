## Markdown

使用标准 Markdown 语法显示图像或视频。

### 图像

::code-preview
![Nuxt 社交图像](https://nuxt.com/new-social.jpg)

#code
```mdc
![Nuxt 社交图像](https://nuxt.com/new-social.jpg)
```
::

或使用本地图像

::code-preview
![日落时云海中的雪山](/mountains.webp)

#code
```mdc
![日落时云海中的雪山](/mountains.webp)
```
::

::note{to="https://image.nuxt.com/"}
Docus 将在底层使用 `<NuxtImg>` 组件而不是原生 `img` 标签。
::

### 嵌入

您也可以使用不同类型的嵌入，例如 YouTube、Twitter 或 CodePen。

::code-preview
```mdc
:u-card
---
to: https://example.com
---

#title
示例卡片
#description
这是一个示例卡片的描述。
::
```
